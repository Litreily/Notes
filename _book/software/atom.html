Atom

Install Packages

安装路径

cd ~/.atom/packages/

安装方法

git clone https://github.com/<user-name>/<package-name>.git
npm install <package-name>

启用方法

快捷键Ctrl+Shift+P打开指令窗口，输入：

Window:reload

或者直接用快捷键Ctrl+Shift+F5重启Atom生效。

常用插件

  • Active-power-mode
  • Minmap
  • git-plus
