# Visual Studio Code

## 常用快捷键

* 指令窗口：`Ctrl+Shift+P`
* 最近文件：`Ctrl+P`
* 选择所有与当前所在单词相同的单词：`Ctrl+F2`
* 选择所有与当前所选内容相同的字符串：`Ctrl+Shift+L`

* 快捷键列表：https://code.visualstudio.com/shortcuts/keyboard-shortcuts-linux.pdf
