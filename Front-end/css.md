# CSS

## Customize chrome scrollbar styles

``` scss
::-webkit-scrollbar {
    width: 5px;
    height: 5px;
}

::-webkit-scrollbar-thumb {
    border-radius: 3px;
    background-color: #888;
}

::-webkit-scrollbar-thumb:hover {
    background-color: #ccc;
}
```
