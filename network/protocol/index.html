    ARP¶

    HW type Protocol type HW size Protocol size Opcode Sender MAC Sender IP Target MAC Target IP
    2 2 1 1 2 6 4 6 4

    说明：

    • HW代表硬件(Hardware)
    • Opcode: 1 → ARP请求，2 → ARP应答，3 → RARP请求，4 → RARP应答

    网络层¶

    IPv4¶

    0~15 16~31
    4位版本 4位首部长度 8位服务类型 16位总长度（字节）
    16位标识 3位标志 13位片偏移
    8位生成时间（TTL） 8位协议 16位首部校验和
    32位源IP地址
    32位目的IP地址
    32位选项（若有）
    数据

    说明：

    • 8位协议: 1 → ICMP ，2 → IGMP ，6 → TCP ，17 → UDP
    • 首部校验和：先将校验和置零，然后以16bit为一个单元，对首部所有单元进行反码求和，结果存入校验和；接收端进行校验时同样对首部进行反码求和，求和结果应当为全1。

    0~15 16~31

    数据（若有）
